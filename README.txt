Stopwatch Web App
-----------------
Files:
- index.html : Main HTML file.
- styles.css : Styles for the UI.
- script.js  : Stopwatch logic (start, pause, reset, lap).
How to use:
1. Unzip the folder.
2. Open index.html in a browser (no server needed).
Keyboard shortcuts:
- Space : Start / Pause
- L     : Record lap
- R     : Reset
